# Hydration_RAG_Assistant
Built a Retrieval-Augmented Generation (RAG) assistant using FAISS and Sentence Transformers to answer hydration-related queries from PDF documents. Implemented semantic search, optimized chunking strategies, and integrated Google Gemini API for context-aware responses via a Streamlit interface.
